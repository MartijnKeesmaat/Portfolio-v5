Introduction

Random Walker

In the introduction we learn about various ways of generating random numbers. Randomness plays a large role in creative coding, giving each render a unique outcome.

A random walker is an entity that follows a path based on a randomly generated input. This one simply draws a single point moves 5 steps and draws another.

Chapter 1

Vectors

The second chapter teaches us about vectors. Vectors are a basic building block in creative coding. Some people explain it very complex. I just see it as something that groups values. Instead of having multiple variables for position, you can group those into one vector. - position(x, y)

The experiment you see here is fairly simple. It calculates the distance of a particle between it and the position of the cursor. Then subtract that a bit for each frame. Repeat that for 50 particles and you get something like this.

Chapter 2

Forces

In the past chapters we moved objects by updating the position with velocity and acceleration. However, the limiting factor was that these values were arbitrary. With forces we give some reason for those values. Like you see here with gravitational forces.

This one is more complex than the previous one as we use two clasess which need to communicate with each other. The attractor class and the particle class. The particle is passed as an argument to the attractor.

Chapter 4

Particle system

A particle system it a system of many objects. Each object sharing similar attributes. In this chapter we learn how to maintain those systems. Here we introduce a lifecycle for each particle to keep things running smoothly.

The experiment that is shown here isn't that fancy. It's main purpose is to demonstrate an infinite source of generated particles which still runs smoothly. This is because of the introduction of the lifecycle.

Chapter 6

Autonomous Agents

This is the chapter where I learned all the things to make the sketch I wanted. During the chapter you learn about various behaviors. This example has seek, flee and arrive behaviors.

This type of effect has calle my name for years and I never really knew how to make it. So with this project I took a big risk and spend 2 out of 3 weeks researching.
