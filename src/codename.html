Sans language

This one is a more personal project. The concept is the lost of a passion. It's a horrible empty feeling. This project was like therapy. By working on it I refound parts of my passion. It was created after a while of not feeling able to do anything. The title is inspired by this: "without language there is no thought". Without is replaced with the french word "sans" also from sans-serif.

I reflected on the projects which used to give me energy by writing. Writing has been a great way explore those thoughts. Which is also why type is an important in the project. The answer was project a like this. Combining visual design with creative coding, talking about something that is personal. Using large type, interesting layouts and unique interactions.
